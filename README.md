# To do list tracker
Pokémon based memory game where clicking a card increase your score and clicking the same card twice ends the game.

## Features
- Dynamic cads from PokéAPI
- Card shuffling system where a click on the card shuffles the entire deck
- Score system that include score and a persistent high-score(best-score)

## What I Learned
- React routing
- Animation library called "framer-motion"

## Tech Stack
- React
- Vite
- CSS
